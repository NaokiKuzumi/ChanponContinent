ちゃんぽん大陸 ver0.09

ちゃんぽん大陸は投稿に特化したシンプルなtwitterクライアントです。
Objective-C2.0,Cocoaを用いて作られているためMacOSXとの相性が非常によいです。

MacOSX+AquaSKKで投稿力の変わらないただひとつのtwitterクライアントーーそれがちゃんぽん大陸


How to use
	Ctrl+Enterで投稿 残り文字数が赤くなったら失敗
	Command+,(メニューの「設定」)で認証やその他設定
	twitter側の設定で認証解除しても認証OKって勘違いしたままなので気をつけてね
	残り文字数クリックでタイトルバーの表示、非表示変更
	タイトルバー非表示でも下部をクリックしてドラッグすればウィンドウ動かせるでよ


感想や善良な要望は@kudzu_naokiまで。


ちゃんぽん大陸は大部分がOAuthConsumer及びMGTwitterEngineによって支えられています。
