ちゃんぽん大陸 ver0.06

ちゃんぽん大陸は投稿に特化したシンプルなtwitterクライアントです。
Objective-C2.0,Cocoaを用いて作られているためMacOSXとの相性が非常によいです。

MacOSX+AquaSKKで投稿力の変わらないただひとつのtwitterクライアントーーそれがちゃんぽん大陸


How to use
	Ctrl+Enterで投稿
	Command+,(メニューの「設定」)で認証や透明度変更
	残り文字数カウントクリックでタイトルバーの表示、非表示変更



ちゃんぽん大陸は大部分がOAuthConsumer及びMGTwitterEingineによって支えられています。
http://code.google.com/p/oauthconsumer/
http://mattgemmell.com/source
